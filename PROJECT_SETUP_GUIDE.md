## 1. WHAT TYPE OF SHOPIFY STORE IS THIS?

### Store Classification: **Service-Based Rental Platform**

This is a **hybrid service + product store** that combines:

| Component | Type | Description |
|-----------|------|-------------|
| **Primary Business** | Service Rental | Hourly room/space rentals (Podcast Room, White Wall Studios) |
| **Revenue Model** | Credit-Based | Bundles sold as products → Credits → Bookings |
| **Additional Services** | Service Add-Ons | Coaching, production services, post-production |
| **Physical Products** | Optional | Printing services (business cards, flyers) |

### Shopify Store Category:

**Best Fit:** **"Service Marketplace"** or **"Rental & Booking Platform"**

**Similar to:**
- Co-working space booking platforms
- Studio rental services
- Event space rentals
- Photography studio bookings
- Workshop/class booking with product sales

**Not:**
- ❌ Traditional e-commerce (selling physical products only)
- ❌ Subscription box service
- ❌ Digital downloads store
- ❌ Dropshipping store

## 3. FREE TRIALS & GETTING STARTED

### 3.1 Shopify Free Trial

**Current Offer:**
- **3-day free trial** (no credit card required)
- **$1/month for 3 months** promotional rate (after trial)
- **Total: ~90 days** of low-cost access

**What You Can Do During Trial:**
- ✅ Install Dawn theme
- ✅ Create all pages
- ✅ Add products (bundles)
- ✅ Configure navigation
- ✅ Set up payment methods
- ✅ Test checkout (test mode)
- ✅ Embed SimplyBook.me widget
- ✅ Configure email templates
- ✅ Build store structure

**Limitations:**
- ⚠️ Can't process real payments (until paid plan)
- ⚠️ Limited to test mode
- ⚠️ Some advanced features locked

**After Trial:**
- Choose plan: **Basic ($29/mo)** or **Shopify ($79/mo)**
- All your work is preserved
- Go live immediately

**Recommendation:**
- Start with free trial
- Build entire store structure
- Test integrations
- Upgrade when ready to go live

## 6. DOMAIN NAME CONSIDERATIONS

### Domain Extension Priority:

1. **.com** (Best - most trusted)
2. **.co** (Good alternative)
3. **.io** (Tech/startup feel)
4. **.studio** (Fits your business)

### Domain Availability Check:

Before finalizing project name, check:
- Domain availability (.com preferred)
- Social media handles (Instagram, Twitter, Facebook)
- Trademark conflicts
- Similar existing brands

**Tools:**
- Namecheap.com (domain search)
- Namecheckr.com (social media check)
- USPTO.gov (trademark search)
